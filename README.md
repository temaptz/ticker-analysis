# Ticker Analysis

**AI-платформа для прогнозирования цен, комплексной оценки новостного фона и принятия решений через LangGraph.**

---

## Ключевые возможности

### ML-модели прогнозирования и рейтинга активов
Ансамбль CatBoost моделей покрывает краткосрочные и среднесрочные горизонты, а также агрегирует оценку привлекательности покупки/продажи:
- **TA-3_volume** — классификация трафика объёмов и RSI (CatBoostClassifier) для локального импульса объёма
- **TA-3_tech** — регрессия краткосрочного движения цены (1-20 дней) на основе 20 дневных свечей и техиндикаторов
- **TA-3_fundamental** — классификация направления (upper/same/lower) по ключевым фундаментальным переменным
- **Consensus модели** — CatBoost-агрегаторы рейтингов всех агентов, формирующие обобщённую привлекательность покупки и продажи
- **TA-2_1** — регрессия долгосрочного изменения цены (до года) на 95 признаков: технический стек, фундаментал, consensus-прогнозы, новостной фон и исторические экстремумы

### LLM-анализ и LangGraph для оценки влияния
LLM pipeline включает две независимые подсистемы по оценке новостей:
- **YandexGPT / yandexgpt-lite** — few-shot классификация новости по трем метрикам (тон, сила влияния, фокус упоминания) перед кэшированием
- **LangGraph News Rank (news_rate_v2)** — граф: sentiment → impact_strength → mention_focus → structured response, результаты ложатся в `news_rate_2` и `influence_score`

### LangGraph Decision Graph, MCP и Backtest
Граф LangGraph не выпускает торговые заявки сам по себе, а формирует общий рейтинг привлекательности покупки и продажи. Он — связующее звено между ML и AI-интеграцией:
- **LangGraph Decision Graph** собирает сигналы 8 агентов (MACD, RSI, Tech, News, Fundamental, Volume, Profit, Consensus) и взвешивает их по настраиваемым `weights`; итог — прозрачный `total_rate` для UI и аналитики.
- **MCP Server** — FastMCP-прокси, через который внешние AI (Claude, Cursor и др.) запрашивают рейтинги, прогнозы, новости и генерацию заявок как структурированный LangGraph-ответ; это следующий по значимости слой после ML и LLM.
- **Backtest** — исторический анализ тех же рейтингов, который делает обратную связь по `total_rate` и проверяет, насколько эти сигналы отражают движение активов.

## ML-модели

### TA-2_1: Основная регрессионная модель

**95 признаков:**
- Категориальные: название, валюта, страна
- Технические: RSI, MACD, объём
- Фундаментальные: P/E, EPS, EBITDA, Debt/Equity
- Прогнозы аналитиков: consensus
- Новостной фон: 5 недель influence_score
- Исторические: min/max изменения за 2d...3y

**Гиперпараметры:**
```
iterations: 10000
learning_rate: 0.03
depth: 8
l2_leaf_reg: 5
early_stopping_rounds: 300
```

### TA-3_tech: Краткосрочный прогноз

**203 признака** для 20 дневных свечей:
- OHLCV данные (нормализованные)
- EMA50, RSI, MACD histogram
- Influence score новостей

**Горизонт:** 1-20 дней

### Consensus модели

Агрегация 7 аналитических агентов в итоговый рейтинг (0-1):
- Обучение на исторических данных
- Настраиваемые веса через UI

Подробнее: [ML_MODELS.md](./doc/ML_MODELS.md)

## LLM-анализ новостей

### LangGraph News Rank

```
START → sentiment_rate → impact_strength_rate → mention_focus_rate → END
```


|     Метрика     | Диапазон | Описание             |
|-----------------|----------|----------------------|
| Sentiment       | -1...+1  | Тон новости          |
| Impact Strength |  0...1   | Сила влияния         |
| Mention Focus   |  0...1   | Фокус на организации |



```python
influence_score = sentiment * impact_strength * mention_focus
```

## Особенности

- **Локальный LLM** — полная приватность данных, no API costs
- **LoRA-адаптеры** — дообучение под финансовую доменную область

- **Ансамбль моделей** — устойчивость к переобучению
- **Real-time backtest** — валидация на исторических данных
- **MCP Server** — интеграция с AI-ассистентами (Claude, Cursor)
